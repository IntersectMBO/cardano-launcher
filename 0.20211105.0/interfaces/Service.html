stop

    Stops the process.

    If the process doesn't exit within timeoutSeconds, it will be killed. The default timeout is defaultTimeoutSeconds.

    Parameters

    • Optional timeoutSeconds: number

      How long to wait for the service to stop itself before killing it. If 0, any running timeout kill be cancelled and the process killed immediately.
